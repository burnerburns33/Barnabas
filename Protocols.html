VPN Protocols

Tunnels are a core crucial part of VPN; they serve as the foundation for VPN. These tunnels are based on the following protocols:

  • L2LTP
    • Installed with TCP/IP, both the VPN client and the server are required. L2TP has encryption services using IPsec.
  • PPTP
    • The VPN Server allows two interfaces, one on the internet and the other on a private network. This allows the IP header to be encrypted and sent to the internet.
  • SSTP
    • An unusual tunneling protocol that uses the HTTPS protocol on TCP port to send traffic around firewalls that can block PPTP. The SSL channel of the HTTPS protocol is used to encrypt this message.
